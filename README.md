# Boot-Camp-Exercises

Word Count

The module WordCnt implements a function words that accepts a single string parameter and returns a dictonary
with each unique word as the key and the number of its occurences as the value.

Max and Min

The maxmin module implements find_max_min that takes a list as its only argument and returns retuns a list
having the smallest and largest elements.

Tweets

The tweets.py module is run as a script and allows one to make a tweet.
For the script to run,the twitter library is needed and can be downloaded for free from the link below.
https://pypi.python.org/pypi/twitter

ALso, to learn how to use another account instead of the account provided in the script, click on the link below.
